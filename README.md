## Ejecutar localmente

Clonar el proyecto

```bash
  git clone https://github.com/julandrod/Gemini-Chatbot.git
```

Ir al directorio del proyecto e instalar paquetes en el frontend

```bash
  cd Gemini-Chatbot
  cd frontend
```

Instalar las dependencias

```bash
  npm install
```

Ejecutar el servidor

```bash
  npm run dev
```

### Repetir con el backend

```bash
  cd Gemini-Chatbot
  cd backend
```

Instalar las dependencias

```bash
  npm install
```

Ejecutar el servidor

```bash
  npm run dev
```
